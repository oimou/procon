数列$\{B_n\}$を次のように定義する：
\begin{equation}
    B_n = \mathrm{max}\{A_{n + 1} - A_n - T, 0\} \hspace{15pt} (1 \leq n \leq N - 1)
\end{equation}
$B_n$は，$n + 1$人目と$n$人目の間でドアが開いていない秒数に一致する。
したがって，求める時間は，
\begin{equation}
    A_N + T - \left(A_1 + \sum_{i = 1}^{N - 1} B_i \right)
\end{equation}
である。
